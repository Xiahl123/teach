# 提交私人仓库
1.添加远程仓库:git remote add **** xxxxxx
****代表远程仓库名称
xxxxxx代表远程仓库的ssh地址
提交私人仓库:git push -u -f **** master
****代表远程仓库的别名
# 拉取私人仓库
git pull github 之后输入rsa证书密码

# 使用token访问私有仓库
